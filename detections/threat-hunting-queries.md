# Threat Hunting Queries

## SSH Brute Force Detection

```text
rule.description:"sshd: authentication failed."
```

Detects repeated failed SSH authentication attempts.

---

# MITRE ATT&CK Events

```text
rule.mitre.id:*
```

Displays events mapped to MITRE ATT&CK techniques.

---

# High Severity Alerts

```text
rule.level:[10 TO 15]
```

Displays high severity security events.

---

# Privilege Escalation Detection

sudo

# Persistence Detection

```text
useradd
```

# Source IP Investigation

```text
data.srcip:"192.168.100.20"
```

Displays events generated by the attacker machine.

# SSH Successful Login
