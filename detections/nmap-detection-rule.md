# Nmap Scan Detection Rule

## Objective

Detect possible reconnaissance activity generated by Nmap scanning.

---

# Detection Logic

The rule detects logs containing:

```text
Nmap scan detected
```
---

# MITRE ATT&CK Mapping

|    Technique    |   ID  |
|-----------------|-------|
| Active Scanning | T1595 |

---

# Rule Configuration

```xml
<group name="nmap_detection,syslog,">

  <rule id="100200" level="10">

    <decoded_as>syslog</decoded_as>

    <if_sid>1002</if_sid>

    <match>Nmap scan detected</match>

    <description>Possible Nmap reconnaissance activity detected</description>

    <mitre>
      <id>T1595</id>
    </mitre>

  </rule>

</group>
```
